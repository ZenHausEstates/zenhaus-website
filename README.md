# ZenHaus Estates Website (Next.js App Router)

**Base:** Venice, FL • **Service Area:** Southwest Florida (Tampa Bay → Naples)

## Quick Start
```bash
npm install
npm run dev
# build static export for Vercel/Netlify
npm run build
```
Output in `out/`.

## Edit Content
All copy lives in `content/content.json`.

## Forms
Static forms compatible with Netlify/Vercel detection:
- seller-intake
- partner-intake
- contact
